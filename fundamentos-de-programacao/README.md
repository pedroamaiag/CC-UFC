# Fundamentos de Programação

Este diretório reúne exercícios e práticas da disciplina de Fundamentos de Programação.

Todas as listas de exercícios dessa disciplina foram originalmente propostas em Python, mas eu adaptei as soluções para C++. A ideia foi aproveitar os mesmos problemas para praticar uma linguagem que estou estudando e que quero dominar melhor, tanto pela sua importância em disciplinas como Cálculo Numérico quanto pela sua relevância na programação competitiva.
## Objetivo

O objetivo deste material é reforçar conceitos básicos de programação, como:

- funções;
- laços de repetição;
- estruturas condicionais;
- expressões booleanas;
- recursão;
- vetores;
- matrizes;
- manipulação de arquivos;
- structs;
- entrada e saída de dados;
- operações matemáticas;
- organização de código em C++.
